## 🧪 Labs Covered

Each lab is designed to simulate a real-world SQLi scenario. Here's a non-exhaustive list of labs included (check the `Labs/` folder for full details):

1. **Lab 1** – SQLi in WHERE clause
2. **Lab 2** – SQLi with Login Bypass
3. **Lab 3** – Union-based SQLi
4. **Lab 4** – Error-based SQLi
5. **Lab 5** – Blind SQLi (Boolean and Time-based)
6. **Lab 6** – Out-of-Band SQLi
